# Contributing to DepUp npm packages

Thanks for your interest in the project! To get started:

1. Install dependencies with `npm install`.
2. Generate a scoped clone of a package using `npm run depup -- <package-name>`.
3. Generated packages are written to `<name>/<version>/rev-<n>`.
4. Submit pull requests with your changes. Please run available checks before submitting (currently `npm test`).

We welcome ideas and improvements via issues and pull requests.
